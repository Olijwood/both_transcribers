About

This is Transcription Service uses a Python backend running both Whisper(OpenAI) and DeepSpeech(Mozilla) to transcribe your audio files. Your file is formatted with FFMPEG so it can be correctly handled by each AI transcriber. Once your file is in the correct format, it is split into chunks so that it becomes more manageable for the Transcriber AI's to work with. This is done simultaneously using Ray Parallel processing.


Once the file is transcribed by both of the AI's you will be redirected to a form where you can choose a title (default being the file name) and edit the content of the transcription. This is the case as it's not perfect so you may wish to make some adjustments. Once you have clicked 'Save', you will be able to view the transcription in your 'Transcriptions' section in your account page.
